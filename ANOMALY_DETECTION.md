# 异常检测功能说明 - 2025-10-31

## 功能简介

异常检测系统基于**统计学理论**和**百家乐理论概率**，自动识别异常靴牌，避免在异常情况下产生不可靠的预测。

## 理论基础

### 百家乐理论概率（8副牌）

```
庄家赢: 45.86%
闲家赢: 44.62%
和局:    9.52%
```

### 连续概率

```
连20次: 0.0000007% (1/140,000,000)
```

## 检测维度

### 1. 统计显著性检验（Z-score）

**原理**：检验庄闲比例是否显著偏离理论值

```python
Z = (实际值 - 理论期望) / 标准差

判断标准：
Z > 2.58  → 异常（99%置信度, p<0.01）
Z > 3.29  → 严重异常（99.9%置信度, p<0.001）
```

**示例**：
```
正常: 60局中庄30次 → Z=0.02 ✅
异常: 60局中庄45次 → Z=3.9 ❌ (概率<0.01%)
```

---

### 2. 极端长连检测

**判断标准**：
```
连12次及以上 → 异常（理论概率<0.01%）
连15次及以上 → 严重异常（理论概率<0.001%）
```

**示例**：
```
正常: 最长连续5次 ✅
异常: 庄连12次 ⚠️
严重: 庄连18次 ⛔
```

---

### 3. 和局比例异常

**判断标准**：
```
和局>15%  → 异常警告
和局>20%  → 严重异常
和局<3%   → 异常警告（过少也异常）
```

**示例**：
```
正常: 和局9.5% ✅
异常: 和局18% ⚠️
严重: 和局35% ⛔
```

---

### 4. 卡方检验（整体分布）

**原理**：检验整体分布是否符合理论

```python
χ² = Σ [(观察-期望)² / 期望]

临界值（自由度=2）：
χ² > 9.21   → 异常（p<0.01）
χ² > 13.82  → 严重异常（p<0.001）
```

## 异常等级

### 评分系统

```
严重度评分 = Σ(各项检测的严重度分数)

各项分数：
- 统计Z-score严重异常: +4
- 统计Z-score异常: +2
- 极端长连(≥15): +5
- 长连异常(≥12): +3
- 和局严重过多(>20%): +3
- 和局异常(>15%): +2
- 卡方严重异常: +3
- 卡方异常: +2
```

### 等级划分

| 评分 | 等级 | 说明 | 置信度调整 |
|------|------|------|-----------|
| 0 | normal | 正常 | 100% |
| 1-2 | low | 轻微异常 | 80% |
| 3-5 | moderate | 中度异常 | 60% |
| 6-9 | high | 高度异常 | 40% |
| 10+ | critical | 严重异常 | 20% |

## 系统响应

### 自动调整置信度

```python
if 异常评分 >= 10:
    置信度 × 20%  # 降至原来的20%
elif 异常评分 >= 6:
    置信度 × 40%  # 降至原来的40%
elif 异常评分 >= 3:
    置信度 × 60%  # 降至原来的60%
elif 异常评分 > 0:
    置信度 × 80%  # 降至原来的80%
```

### 用户建议

```
严重异常(10+): "⛔ 强烈建议停止预测！靴牌极度异常，可能存在问题"
高度异常(6-9): "🚫 建议谨慎！靴牌高度异常，预测可靠性极低"
中度异常(3-5): "⚠️ 注意风险！靴牌存在明显异常，建议降低投注"
轻微异常(1-2): "💡 轻微异常，适当谨慎"
正常(0):       "✅ 靴牌正常"
```

## 优势

### ✅ 优点

1. **基于科学理论**
   - 统计学显著性检验
   - 百家乐理论概率验证

2. **自动化保护**
   - 无需用户判断
   - 自动调整置信度

3. **多维度检测**
   - 庄闲比例
   - 长连情况
   - 和局分布
   - 整体拟合度

4. **分级响应**
   - 轻微异常→提示
   - 严重异常→强烈警告

5. **透明度高**
   - 显示具体异常项
   - 说明检测原理
   - 量化评分

### ⚠️ 局限性

1. **需要足够数据**
   - 至少30局才检测
   - 数据越多越准确

2. **不能100%确定**
   - 统计显著≠绝对错误
   - 低概率事件仍可能发生

3. **无法检测所有异常**
   - 只检测统计异常
   - 无法检测人为操纵

## 总结

异常检测系统通过**4个维度**的统计检验，识别偏离理论概率的异常靴牌，并自动**降低预测置信度**或**建议停止预测**，有效保护用户避免在异常情况下做出错误判断。

这是一个**低风险、高价值**的功能，不影响正常预测，只在异常时发挥作用。

**使用建议**：
- ✅ 看到轻微异常：适当谨慎
- ⚠️ 看到中度异常：降低投注
- ⛔ 看到严重异常：停止预测，检查数据
