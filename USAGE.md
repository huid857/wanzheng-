# 使用指南

## 快速开始

### 第一步：安装依赖

```bash
pip install -r requirements.txt
```

如果安装TensorFlow遇到问题，可以尝试：

```bash
# CPU版本（推荐，更轻量）
pip install tensorflow-cpu

# 或者，如果TensorFlow安装失败，可以跳过
# 系统会自动禁用LSTM模型，其他模型仍可正常使用
```

### 第二步：运行程序

```bash
python main.py
```

## 使用场景

### 场景1：实时记录（边玩边用）

1. 启动程序，选择 `1. 实时记录模式`
2. 每开一局，输入结果：
   - 输入 `B` 或 `庄` = 庄家赢
   - 输入 `P` 或 `闲` = 闲家赢  
   - 输入 `T` 或 `和` = 和局
3. 输入 `?` 随时获取预测
4. 输入 `.` 查看当前数据和路单
5. 输入 `u` 撤销上一局（如果输错了）

**示例流程：**

[10局] 输入结果: ?

🎯 预测结果

```

```

3. 输入空行结束

### 场景3：查看详细分析

选择 `5. 查看详细分析` 可以看到：

- 📊 路单图（大路）
- 📝 最近结果列表
- 📈 详细统计（庄闲和的频率、最长连胜等）
- 🎯 综合预测
- 🤖 各个模型的预测详情
- 💰 投注建议

## 功能说明

### 1. 路单显示

系统会自动生成路单图（大路）：

```
● ● ●     ● ●
●         ● ●
          ●

● 庄  ● 闲
```

红色代表庄，蓝色代表闲。

### 2. 预测模型

系统使用多个模型进行集成预测：

#### 基础模型（始终启用）：
- **N-gram分析**：统计历史模式（如"庄庄闲"后出什么）
- **马尔可夫链**：基于状态转移概率
- **趋势分析**：滑动窗口统计
- **连胜分析**：跟龙/斩龙策略
- **频率统计**：简单但有效

#### 高级模型（需要足够数据）：
- **LSTM神经网络**：需要300+局数据
- **随机森林**：需要200+局数据

### 3. 置信度说明

- **强烈推荐（80%+）**：多个模型高度一致
- **推荐（60-80%）**：模型较为一致
- **谨慎参考（<60%）**：模型分歧较大或数据不足

### 4. 数据积累

系统会自动保存历史数据：

- 每靴牌可以单独保存
- 所有历史数据会合并用于学习
- 数据越多，预测越准（理论上）

**建议：**
- 第1靴：60局（基础预测）
- 5靴以上：300+局（启用高级模型）
- 15靴以上：900+局（最佳效果）

## 常见问题

### Q1: 数据太少怎么办？

A: 系统在数据较少时（<10局）也能工作，但会提示"置信度低"。建议至少记录10-20局再开始使用预测功能。

### Q2: 中途进入怎么办？

A: 没问题！从您进入的那一局开始记录即可，不需要知道之前的局数。

### Q3: 不知道点数、补牌怎么办？

A: 完全不需要！系统只需要最终结果（庄/闲/和），不需要点数、牌面等信息。

### Q4: 一靴牌打完了怎么办？

A: 选择 `6. 开始新靴牌`，系统会询问是否保存当前靴牌。保存后，这些数据会加入历史库，用于未来的预测。

### Q5: TensorFlow安装失败？

A: 没关系！LSTM只是多个模型之一。即使不安装TensorFlow，系统仍有5-6个其他模型正常工作。

### Q6: 预测准确率如何？

A: **重要提示**：百家乐理论上是独立随机事件，任何预测系统都无法保证准确率。本系统仅供学习研究，不保证盈利。请理性娱乐。

## 数据管理

### 导出数据

选择 `7. 数据管理` > `1. 导出当前靴牌数据`

会生成一个文本文件，如：
```
BBPPTBPPBPBPBBBPPTBPPBP
```

可以备份或分享。

### 导入数据

选择 `7. 数据管理` > `2. 从文件导入`

输入文件路径即可导入。

### 查看历史统计

选择 `7. 数据管理` > `3. 查看历史统计`

可以看到：
- 总共记录了多少局
- 多少靴牌
- 整体庄闲和的分布
- 是否达到启用ML模型的数据量

## 高级技巧

### 技巧1：每10局查看一次

实时记录模式下，系统会在第10、20、30...局自动显示预测，帮助您把握节奏。

### 技巧2：对比多个模型

选择查看"各模型预测详情"，可以看到每个模型的预测和理由。如果模型高度一致，置信度更高。

### 技巧3：注意连胜情况

系统有专门的"连胜分析"模型：
- 连2-3次：跟龙策略
- 连5次以上：预测断龙

### 技巧4：累积数据

每次玩都保存靴牌数据，系统会越来越"聪明"（理论上）。
