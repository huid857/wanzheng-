# 百家乐智能预测分析系统 🎰

## 项目简介

这是一个基于机器学习和统计分析的百家乐预测系统，能够从历史数据中学习模式，并对下一局进行预测。

## 核心功能

- ✅ **超简单数据输入**：只需输入 B/P/T（庄/闲/和）
- 🧠 **多维度分析**：N-gram模式、路单分析、趋势统计
- 🤖 **机器学习预测**：LSTM神经网络、马尔可夫链、随机森林
- 📊 **可视化展示**：路单图、概率分布、趋势图表
- 💾 **数据积累**：跨靴牌学习，越用越智能

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 运行程序

```bash
python main.py
```

### 3. 使用方法

#### 方式一：实时记录模式
- 启动程序后选择"实时记录"
- 每开一局，输入结果（B=庄, P=闲, T=和）
- 10局后系统开始预测

#### 方式二：批量导入模式
- 准备历史数据（如：BBPPTBPPBPBP）
- 选择"批量导入"
- 粘贴数据即可

## 数据格式

支持多种输入格式：

```
# 格式1：字母
BBPPTBPPBPBP

# 格式2：中文
庄庄闲闲和庄闲闲庄闲庄闲

# 格式3：逗号分隔
B,B,P,P,T,B,P,P,B,P,B,P

# 格式4：每行一个
庄
庄
闲
闲
和
```

## 预测模型

系统使用多个模型进行集成预测：

### 核心预测模型（8个）

1. **历史模式集成 (Historical)** ⭐ **优化**
   - 内部集成：N-gram序列分析 + 马尔可夫链 + 数据驱动统计
   - 避免冗余投票，提高预测准确性
   - 基于真实历史数据，靴内统计分析

2. **当前趋势分析 (Trend)**
   - 分析最近10-20局的走势
   - 识别庄强/闲强/平衡态

3. **连胜模式分析 (Streak)**
   - 分析当前连胜状态
   - 智能判断跟龙/断龙时机

4. **频率分布 (Frequency)**
   - 当前靴牌的庄闲和频率
   - 反映实际概率分布

5. **LSTM神经网络（原始）**
   - 深度学习序列预测
   - 数据量>300局时启用

6. **随机森林（原始）**
   - 基于手工特征的分类
   - 8个简单特征

7. **LSTM V2（特征工程）** 🆕
   - 使用12个精选特征的LSTM
   - 准确度提升+3.79%
   - 特征包括：连续、趋势、波动、周期等

8. **RandomForest V2（特征工程）** 🆕
   - 使用8个优化特征的随机森林
   - 经过600+配置深度优化
   - 准确度提升+6.65%

### 模型优化（2025-10-30）⭐
- ✅ **解决冗余问题**：合并相似模型，避免过拟合
- ✅ **投票平衡**：历史统计 vs 当前趋势 权重更合理
- ✅ **透明度提升**：清晰显示各模型贡献

### 特征工程（2025-10-31）🎯 **最新**
- ✅ **25个候选特征**：覆盖频率、连续、交替、趋势、周期、数据、变化率7大类
- ✅ **科学评估方法**：互信息+随机森林+置换重要性 三重评估
- ✅ **精选12特征（LSTM V2）**：current_streak_length, banker_change_rate, volatility等
- ✅ **精选8特征（RF V2）**：深度优化，600+配置测试
- ✅ **相关性检查**：自动移除高度相关特征（>0.8）
- ✅ **准确度提升**：LSTM V2 +3.79%, RF V2 +6.65%

### 异常检测（2025-10-31）🔍
- ✅ **统计显著性检验**：Z-score检测庄闲比例异常
- ✅ **极端长连检测**：识别统计不可能的长连
- ✅ **和局异常检测**：检测和局比例偏离
- ✅ **卡方检验**：整体分布拟合度检验
- ✅ **自动置信度调整**：异常情况自动降低置信度
- ✅ **分级警告**：轻微/中度/高度/严重 4级响应

### 高级靴牌分析（阶段3 - 2025-10-30新增）⭐
- ✅ **靴牌类型识别**：自动识别长龙靴/跳跳靴/平衡靴等6种类型
- ✅ **靴牌阶段分析**：分析前/中/后期特征，把握靴牌节奏
- ✅ **相似靴牌查找**：找出历史相似靴牌，参考走势
- ✅ **动态权重调整**：根据靴牌特征智能调整模型权重
- ✅ **实时显示**：预测界面自动显示靴牌分析信息
