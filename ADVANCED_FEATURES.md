## 实施时间

## 概述
完整实施高级靴牌分析功能，系统现在能够：
1. 自动识别靴牌类型
2. 分析靴牌不同阶段的特征
3. 查找历史相似靴牌
4. 根据靴牌特征动态调整模型权重

---

## 📊 新增功能

### 1. 靴牌类型识别

系统自动识别6种靴牌类型：

#### 主类型（基于庄闲比例）
- **庄强靴** - 庄赢率 ≥ 52%
- **闲强靴** - 闲赢率 ≥ 52%
- **平衡靴** - 庄闲差异 ≤ 4%

#### 特征类型（可多个）
- **长龙靴** - 连续4+次出现 ≥ 3次
- **跳跳靴** - 单跳/双跳 ≥ 60%
- **和多靴** - 和局比例 ≥ 12%

#### 示例输出
```
▸ 靴牌类型: 长龙靴(3个长龙) | 和多靴(13.2%)
  庄: 47.2% | 闲: 39.6% | 和: 13.2%
  长龙数: 3 个
```

---

### 2. 靴牌阶段分析

将靴牌分为三个阶段进行独立分析：

- **前期** (1-20局) - 开局走势
- **中期** (21-40局) - 发展阶段
- **后期** (41-60局) - 收官阶段

#### 示例输出
```
▸ 当前阶段: 中期 (32局)
  前期(1-20局): 平衡靴
  中期(21-40局): 庄强靴(庄58%)
  后期(41-60局): [未到达]
```

---

### 3. 相似靴牌查找

从历史中查找与当前靴牌相似的靴牌（前3名）：

#### 相似度计算因素：
- 庄闲比例相似度 (40%权重)
- 前N局序列相似度 (30%权重)
- 连续特征相似度 (30%权重)

#### 示例输出
```
▸ 相似历史靴牌:
  1. shoe_20251029_011750 (相似度88%) - 长龙靴
  2. shoe_20251030_095150 (相似度81%) - 平衡靴
  3. shoe_20251029_092151 (相似度78%) - 闲强靴
```

**用途：** 参考相似靴牌的后续走势

---

### 4. 动态权重调整

根据靴牌类型自动调整模型权重：

#### 长龙靴
```
Streak权重: × 1.3  (提升连续预测)
Trend权重:  × 0.8  (降低趋势预测)
DataDriven: × 1.2  (提升断龙预测)
```

#### 跳跳靴
```
Streak权重: × 0.6  (大幅降低连续预测)
Trend权重:  × 1.3  (提升趋势预测)
Frequency:  × 1.2  (提升频率预测)
```

#### 庄强/闲强靴
```
Frequency:  × 1.1  (轻微提升频率预测)
```

## 📈 权重调整逻辑

系统现在有**三层权重调整**：

### 1. 基础权重
```python
Trend:       1.5  # 当前趋势
DataDriven:  1.0  # 历史统计
Streak:      0.8  # 连续预测
Frequency:   1.2  # 当前频率
Markov:      1.0
N-gram:      1.0
LSTM:        1.2
RandomForest: 1.2
```

### 2. 历史调整（基于预测准确率）
```python
连续错误2次: 错误模型 × 0.5, 正确模型 × 1.5
连续错误3次: 错误模型 × 0.3, 正确模型 × 2.0
连续错误4次: 错误模型 × 0.1, 正确模型 × 3.0
```

### 3. 靴牌调整（基于靴牌类型）⭐ 新增
```python
长龙靴: Streak × 1.3
跳跳靴: Streak × 0.6, Trend × 1.3
庄强/闲强: Frequency × 1.1
```

### 最终权重计算
```
最终权重 = 基础权重 × 历史调整系数 × 靴牌调整系数
```

## ⚠️ 注意事项

1. **最小数据要求**
   - 靴牌分析需要至少10局数据
   - 相似靴牌查找需要至少3个历史靴

2. **相似度阈值**
   - 默认最小相似度：50%
   - 低于阈值的靴牌不显示

3. **权重调整优先级**
   - 历史调整 > 靴牌调整
   - 连续失败时，历史调整占主导
